# Skills

> Rules for Agent Skills files.

Agent `SKILL.md` files SHOULD always have a H1 heading that is a "prettier" (appropriately capitalized) version of their `name` metadata.

Agent `SKILL.md` files SHOULD contain concise, actionable rules for the agent. Detailed explanations, context, and background for humans SHOULD be placed in a `README.md` file in the same directory.
